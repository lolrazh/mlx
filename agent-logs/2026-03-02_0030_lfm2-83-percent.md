## Bugs & Issues Encountered
1. **Adapter path must be a directory, not a file** — `--adapter-path` expects a dir containing `adapters.safetensors` + `adapter_config.json`. Passing a `.safetensors` file directly fails with "Not a directory" error.
   - **Fix:** Created `adapters-lfm2-t1-best/` dir with the checkpoint + config copied in.
2. **Resumed training overwrites checkpoint names** — mlx_lm restarts iter counter from 1 on resume. `0000100_adapters.safetensors` from resumed run overwrites the original iter 100 checkpoint.
   - **Impact:** Lost original iter 100-700 checkpoints. Only iter 800 preserved (separate filename). Save important checkpoints to separate dirs before resuming.
3. **User corrected my claim about no bf16 LFM2.5** — `LiquidAI/LFM2.5-1.2B-Instruct-MLX-bf16` exists on HuggingFace. I incorrectly said no bf16 was available. Always search before claiming a model doesn't exist.

## Key Learnings
- **0.014 val loss drop = +5 accuracy points.** Val loss 0.480 → 0.466 moved accuracy from 78% → 83%. With 20 val examples, small loss changes matter. Always benchmark accuracy, don't trust val loss.
- **LFM2 ceiling is ~900 iters on 535 examples.** The resumed run plateaued immediately after a small initial improvement. ~900 total iters (val loss 0.466) was the sweet spot. More training = overfitting.
- **Resume overwrites checkpoints.** mlx_lm iter counter resets on resume. The iter 100 save from the resumed run overwrites the original iter 100. Always backup before resuming.
- **LFM2.5-1.2B bf16 = 4-bit at zero-shot.** Both scored 9%. Confirms architecture, not precision, is the zero-shot bottleneck for LFM hybrid models.

## Architecture Decisions
- **LFM2.5-1.2B over Llama 3B for fine-tuning** — Same architecture that hit 83%. Higher-upside experiment: if 1.2B works, it's a 1.2 GB deploy model at sub-1s latency. Llama 3B would just be another ~3B transformer.
- **1000 iters for 1.2B** — LFM2-2.6B needed ~900 for best results. Smaller model might need more passes. 1000 gives headroom.
- **Baseline Llama 3B anyway** — Quick zero-shot test (5 min) while 1.2B trains. Answers whether Llama 1B's failure was about 1B or about Llama.
